# Sales Performance Analysis: From Messy Data to Interactive Dashboard

## 📌 Project Overview
This project demonstrates an end-to-end data analysis workflow, transforming a raw, "messy" sales dataset into a clean, structured, and interactive dashboard. The goal was to provide actionable insights into regional sales performance, achievement percentages, and key growth metrics.

## 🛠️ Technical Stack
* **Data Cleaning & ETL:** Power Query
* **Data Modeling:** Power Pivot (Star Schema)
* **Calculations:** DAX (Data Analysis Expressions)
* **Visualization:** Power BI / Advanced Excel

## 🚀 The Data Journey

### 1. Data Cleaning (The "Messy" Challenge)
The raw dataset was highly unorganized with inconsistent formatting and missing values. I performed the following:
* **Handling Nulls:** Identified and managed missing entries to ensure data integrity.
* **Data Transformation:** Standardized date formats and categorized product/region names.
* **Schema Structuring:** Converted "flat files" into structured tables ready for modeling.

### 2. Data Modeling & DAX
To enable deep-dive analysis, I built a relational model using **Power Pivot**:
* Developed a **Star Schema** for optimized performance.
* Created custom **DAX Measures** to calculate critical KPIs:
    * `Achievement %`: Comparing actual sales vs. targets.
    * `Regional Rankings`: Dynamically ranking performance by territory.
    * `YTD/YoY Growth`: Analyzing performance over time.

### 3. Interactive Visualization
The final dashboard provides a high-level view of:
* Top performing regions and sales representatives.
* Achievement vs. Target tracking.
* Dynamic filtering by date, region, and product category.
